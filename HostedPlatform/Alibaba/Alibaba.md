# 功能列表

阿里巴巴函数计算提供的功能有：

* 丰富的触发器类型
	* 阿里云各种产品触发器，如对象存储、日志服务、消息队列、CDN
	* Http 触发器
	* 时间触发器
* 多种编程语言支持
	* Node.js
	* Python
	* Java
	* PHP
	* C#
	* Custom Runtime
* 便捷的开发工具
	* [funcraft](https://github.com/alibaba/funcraft)：阿里自研的开发、构建、部署工具
	* VSCode 插件
* 丰富的资源类型
 	* 按量实例：根据请求数动态扩缩
	* 预留实例：针对冷启动的绕过方案
* 灵活的计量模式
	* 后付费模型：按实际使用计算资源计费
	* 预付费（包年包月）模型
