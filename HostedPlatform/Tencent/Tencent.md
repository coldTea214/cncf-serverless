# 功能列表

腾讯 SCF 提供的功能有：

* 代码管理
	* 控制台编辑代码：适用于没有外部依赖的业务代码
	* 上传代码 zip 包
	* 通过 COS 管理 zip 包：在云函数中指定代码所在 Bucket 和 文件对象，平台将自动下载函数代码
* 支持多开发环境
	* Python 2.7 & 3.6
	* Node.js 6.10 & 8.9
	* Java 8
	* Php 5 & 7
* 自动伸缩：根据请求量扩缩容
* 事件触发
	* 腾讯云服务事件
	* Http 触发
	* 时间触发
* 监控和日志
